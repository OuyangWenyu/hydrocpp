# hydrocpp

水文水资源（Hydrology and Water Resources）方面利用 C/C++ 做模型model、算法algorithm等科学计算工作所需的基础技能树学习。

目前主要内容包括：

1. 配置C/C++语言科学计算开发环境
2. 基本的C/C++语法学习

仍在慢慢更新，根据实际碰到的实例来记录相关的基本知识。

本repo下的代码如无特别说明，都是在win10下的Ubuntu18.04中执行的（Ubuntu18.04下应该也问题不大）

environment.yml为jupyter编辑器准备的，需要在win10下Ubuntu系统下安装conda（过程参考：[OuyangWenyu/elks](https://github.com/OuyangWenyu/elks)）并执行以下语句来安装jupyter编辑器需要的环境：

```Shell
conda env create -f environment.yml
```
